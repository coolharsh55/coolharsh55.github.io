  1. Choose your product:
    1. Detergent / soaps / shampoo / oil
    2. Balm / ointment / spray / medicine
    3. Cars / bikes / petrol / machinery
    4. deoderant (!!!)
  2. Select a scenario
    1. Normal home, as if bought yesterday
    2. Cocky mountains, valleys
    3. High end area of town
    4. Anywhere with a crowd
  3. Select characters
    1. Bunch of scantily dressed girls
    2. Few jealous dudes
    3. A wife (hot/cold depends on usage!)
    4. One alpha male
  4. Select script
    1. Unbelievable instant action
    2. Product effect multiplied 1000x times
    3. Over the top rajni-isque action
    4. Logic-less scenes with high quality camera
  5. Think of a (stupid) tag line
  6. Add ear-irritating sound effects
  7. Show ad at primetime
