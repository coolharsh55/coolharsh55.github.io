GitHub bot commands

  • Issue: Foo is undefined - create a new issue
  • action harsh: do something - due in 2 weeks
  • close #15 - close issue
  • reopen #15 - reopen issue
  • ghurlbot, list open issues with label foo - Instead of list, you can also say find, look up, get, search, search for. Using for me will return issues assigned to me or user.
  • ghurlbot, delay 20 - don't look up issues again until 20 lines later

scribe config

command: perl scribe.perl [options] < log.txt > minutes.html

options:

  • -implicitContinuations - continue conversation over multiple lines without ...; to still add general lines to the log use (\) backslash at the start of the line
  • -final - make minutes final rather than draft
  • -emphasis - convert ASCII combinations into symbols and formatting
